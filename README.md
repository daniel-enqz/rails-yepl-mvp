## Setup

```shell
 git clone git@github.com:daniel-enqz/rails-simple-airbnb.git
 cd rails-simple-airbnb
 rails db:create db:migrate db:seed:replant
 bundle install
 yarn install
 rails server
```
Open you browser and visit `localhost:3000`
